WELFake Dataset

WELFake is a large, balanced fake-news dataset commonly used for machine learning and deep learning research on misinformation detection. It compiles articles from multiple open-source datasets and applies unified preprocessing to create a clean, reliable corpus.

Dataset Summary

Total samples: ~72,000

Real news: ~35,000

Fake news: ~37,000

Balanced: ✔ Yes (≈50/50)

Columns
Column	Description
title	News headline
text	Full article content
label	0 = Real, 1 = Fake

Why WELFake?

Balanced classes → fairer training

Large-scale → suitable for ML & DL models

Clean, standardized preprocessing

Diverse sources → better generalization

Citation
Verma et al. (2021). WELFake: Word Embedding Learning for Fake News Detection.
